# Article Drafts

Articles in progress before they go live on the blog.

## Structure

```
drafts/
  <slug>/
    article.md        — the draft (markdown, will become .mdx when published)
    images/            — all images for the article
    gen-diagrams.py    — diagram generation scripts (if any)
```

## Workflow

1. **Draft** → write article + gather/create images here
2. **Review** → Sid reviews in #cosmo-wiki or dashboard
3. **Publish** → move to `content/posts/<slug>.mdx` with full frontmatter, images to `public/blog/<slug>/`
