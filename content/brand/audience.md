# Target Audience — Customer Personas

> Updated: 2026-03-16

## Persona: Independent Artist

- **Role:** Musician who self-releases on Spotify, Apple Music, etc.
- **Pain:** Hates posting on social; the algorithm punishes silence but creating content takes forever
- **Want:** Stay visible to fans while spending time in the studio, not on Canva
- **Budget:** $50-200/month for tools
- **Language:** "content calendar," "the grind," "I'd rather be in the studio," "engagement is down"
- **Channels:** Instagram, TikTok, X, Spotify forums, Reddit
- **Objection:** "But will it sound like me? I don't want generic AI content."
- **Convincer:** Show Gatsby Grace example — AI-driven but with authentic artist voice via context files

## Persona: Label Marketing Manager

- **Role:** Handles marketing for 5-20 artists at a small-to-mid label
- **Pain:** Not enough hours or staff to create unique content for every artist
- **Want:** Scale content output without hiring more people or sacrificing quality
- **Budget:** $500-2000/month for tools (team budget)
- **Language:** "release campaign," "content pipeline," "ROI," "streaming numbers"
- **Channels:** Instagram, TikTok, YouTube, LinkedIn (for B2B visibility)
- **Objection:** "We need control over brand voice for each artist."
- **Convincer:** Context files per artist — the agent reads voice/brand docs before creating anything

## Persona: Artist Manager

- **Role:** Manages 1-5 artists, handles everything from booking to marketing
- **Pain:** Marketing is one of ten jobs, and it always gets deprioritized
- **Want:** A system that handles the marketing grind so they can focus on deals and relationships
- **Budget:** $100-500/month
- **Language:** "too many platforms," "content backlog," "my artists need more visibility"
- **Channels:** Instagram, X, LinkedIn, email
- **Objection:** "I don't trust AI to represent my artists."
- **Convincer:** The agent uses artist-specific context — it doesn't guess, it reads your brand docs

## Persona: Music Tech Enthusiast

- **Role:** Developer, producer, or industry professional interested in AI + music
- **Pain:** Wants to explore AI-powered workflows but existing tools are generic
- **Want:** A music-specific AI platform to experiment with and build on
- **Budget:** $0-100/month (may convert to higher tier)
- **Language:** "API," "agents," "MCP," "prompt engineering," "building in public"
- **Channels:** X, GitHub, Discord, LinkedIn, Hacker News
- **Objection:** "Is this just a wrapper around ChatGPT?"
- **Convincer:** Show architecture — multi-model, Skills system, CLI, MCP integration, Trigger.dev workers
