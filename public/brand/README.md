# Brand assets

All logos, word marks, and profile/hero imagery. Favicons and PWA icons live in `public/icons/`.

| File | Description |
|------|-------------|
| **icon-lightmode.svg** | Logo mark (symbol only) for light backgrounds. Use in header, footer, light UI. |
| **icon-darkmode.svg** | Logo mark (symbol only) for dark backgrounds. Use on dark sections or dark mode. |
| **wordmark-lightmode.svg** | Full word mark (“Recoupable” text) for light backgrounds. Use when the symbol alone isn’t enough. |
| **wordmark-darkmode.svg** | Full word mark for dark backgrounds. |
| **pfp-sky-bg.png** | Profile/hero image (e.g. founder or team) with sky background. Use on about, team, or hero sections. |

Use in code via `/brand/<filename>` (e.g. `src="/brand/icon-lightmode.svg"`).
