# Transcripts

This directory holds **all transcripts** from calls used to inform marketing positioning and copy.

## What goes here

- **Engineering team** — internal calls (product, roadmap, positioning)
- **Customers** — discovery, feedback, success stories, objections
- **Leads** — sales calls, demos, qualification calls

## Purpose

Transcripts are **context for marketing work**. Use them when creating or refining:

- GTM funnels and positioning
- Marketing assets (one-pagers, decks, case studies)
- Videos and scripts
- Website and landing page copy
- Lead magnets and gated content
- Blog posts and social posts
- Email sequences and nurture copy

## How to use

- **Humans:** Drop new transcript files here (e.g. `.md`, `.txt`, or dated folders).
- **Agents/AI:** When working on marketing copy, positioning, or GTM assets in this repo, read relevant transcripts for voice-of-customer, pain language, and proof points.

## Naming

Use clear, consistent names so content is easy to find, e.g.:

- `2025-03-18_customer_acme.md`
- `2025-03-17_lead_demo_indie-label.md`
- `2025-03-16_eng_positioning-sync.md`

Optional: group by type in subfolders (`customers/`, `leads/`, `eng/`) if volume grows.
